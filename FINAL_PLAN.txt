NOVUS.AI — 7 EVENTS TO TRACK (JUDGES WILL CHECK THIS)
--------------------------------------------------------------------------------

Install: npm install @novus/analytics (check novus.ai docs for exact package)
Register at: https://novus.pendo.io/register

Track these events in this order:

  // 1. App loaded
  novus.track('session_started')

  // 2. User submits feedback
  novus.track('feedback_submitted', {
    char_count: input.length,
    has_context: !!context
  })

  // 3. Analysis done (after /api/analyze resolves)
  novus.track('analysis_complete', {
    themes_found: themes.length,
    top_score: themes[0].opportunity_score,
    top_theme: themes[0].name
  })

  // 4. User clicks on a theme card
  novus.track('theme_clicked', {
    theme_name: theme.name,
    rank: theme.rank,
    score: theme.opportunity_score
  })

  // 5. Decision screen viewed (also from /api/analyze response)
  novus.track('decision_viewed', {
    build_now: decision.build_now.theme
  })

  // 6. PRD generated (after Call 3)
  novus.track('prd_generated', {
    theme: selectedTheme.name,
    stories_count: prd.user_stories.length,
    has_evidence: true
  })

  // 7. User exports PRD
  novus.track('prd_exported', {
    format: 'copy'  // or 'pdf' or 'saved'
  })

What judges see in Novus dashboard:
  → Full funnel from input to PRD
  → Where users drop off
  → Most clicked theme
  → Export rate

WHAT I BUILT:
ShipReady turns messy customer feedback into evidence-backed product decisions
and engineering-ready specs in under 60 seconds.

THE PROBLEM:
PMs spend hours reading feedback, guessing at priorities, and writing PRDs from
memory. The result: roadmaps based on the loudest voice, not the real signal.

HOW IT WORKS:
1. Paste any feedback (G2 reviews, Slack, tickets, interviews — anything)
2. AI clusters themes with opportunity scores backed by actual customer quotes
3. AI makes explicit BUILD / DELAY / IGNORE decisions with proof
4. One click generates an evidence-backed PRD where every user story links
   to actual customer quotes

THE DIFFERENTIATOR:
Not summaries. Not sentiment. Actual decisions with proof.
Every recommendation is traceable to real customer words.

BUILT WITH:
Next.js 15 + Tailwind + shadcn/ui + Claude API (claude-sonnet-4-6) + Vercel + Novus.ai

WHO IT'S FOR:
PMs, founders, and product teams who are drowning in feedback but starving
for decisions.
